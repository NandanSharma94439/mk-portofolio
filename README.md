# 💼 Personal Portfolio Website

A modern, responsive portfolio website showcasing my software development projects, technical skills, education, and professional experience. The website is designed to provide recruiters and collaborators with an overview of my work and achievements.

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Professional project showcase
- Skills and technology section
- Education and certifications
- Contact information
- Smooth animations and modern UI

## 🛠 Tech Stack

- React
- JavaScript
- HTML5
- CSS3

## 🚀 Getting Started

```bash
git clone https://github.com/NandanSharma94439/portfolio-website.git
cd portfolio-website
npm install
npm run dev
```
